## General info
This project is a simple meditation app.
	
## Technologies
Project is created with:
* HTML5
* CSS3
* JSES6
	
## Setup
To run this project:
1. Open index.html in browser.
2. Pick how many minutes to meditate.
3. Pick one of the two background sounds to meditate to.
